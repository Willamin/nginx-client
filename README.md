# nginx JSON autoindex client

Elm application.

# Requirements

 - Elm
 - Node.js with npm

# Getting Started

```bash
# Install dependencies
$ elm package install
$ npm install

# Compile client for production
$ make
```

# Configuration

 - Please see [cors.conf][cors-conf]
